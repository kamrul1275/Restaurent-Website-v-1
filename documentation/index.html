Introduction

Installation

Local

  1. Unzip the package.
  2. Open dist/index.html in any modern browser.

Hosting

  1. Upload the dist/ folder to your hosting root/public_html.
  2. Ensure mailer/mail.php has correct e‑mail settings (see Forms section).

Customization

Colors

Edit CSS variables in css/restaurant.css:

:root{
  --color-bg:#0f0f10;
  --color-text:#e6e6e6;
  --color-accent:#ffd54a; /* gold */
}

Typography

Update font imports and scale in css/restaurant.css. Example:

h1{font-size:clamp(28px,4vw,48px);line-height:1.2}
h2{font-size:clamp(22px,3vw,32px)}

Logos & Images

Replace files in images/. Provide both .webp and .jpg for compatibility.

Forms (Contact & Book a Table)

Forms submit via AJAX to mailer/mail.php.

1) Basic Setup

  1. Open mailer/mail.php and set $TO_EMAIL to your recipient address.
  2. (Optional) Enable SMTP using PHPMailer (instructions inside the file).

2) Contact Form – example HTML

<form id="contactForm" action="mailer/mail.php" method="POST">
  <input type="hidden" name="form_type" value="contact">
  <input name="name" required>
  <input type="email" name="email" required>
  <input name="phone">
  <textarea name="message" required></textarea>
  <input type="text" name="website" class="hidden" style="display:none"> <!-- honeypot -->
  <button type="submit">Send</button>
</form>

3) Reservation Form – example HTML

<form id="bookForm" action="mailer/mail.php" method="POST">
  <input type="hidden" name="form_type" value="reservation">
  <input name="name" required>
  <input type="tel" name="phone" required>
  <input type="number" name="persons" min="1" max="20" required>
  <input type="date" name="date" required>
  <input type="time" name="time" required>
  <textarea name="message"></textarea>
  <input type="text" name="website" class="hidden" style="display:none"> <!-- honeypot -->
  <button type="submit">Book a Table</button>
</form>

4) AJAX (optional)

document.querySelectorAll('form').forEach(f=>{
  f.addEventListener('submit',async e=>{
    e.preventDefault();
    const res = await fetch(f.action,{method:'POST',body:new FormData(f)});
    const data = await res.json();
    alert(data.message);
    if(data.success) f.reset();
  });
});
